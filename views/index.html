Image Search Abstraction Layer

Freecodecamp back-end api project

Example usage:

hhttps://ripe-impulse.glitch.me/api/imagesearch/lolcats%20funny?offset=10

Searches for images pertaining to "lolcats funny" with an offset of 10.

https://ripe-impulse.glitch.me/api/recentsearches/

Returns recent searches.
